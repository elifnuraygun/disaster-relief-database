# Disaster Relief Database System

This project focuses on designing a relational database system to manage disaster relief operations, including volunteers, aid distribution, and organizations.

## Project Overview
- Relational database design using MySQL
- 3NF normalization for data integrity
- Efficient coordination of volunteers and aid resources

## Key Features
- Structured tables for volunteers, aid, distribution, and organizations
- Complex SQL queries using JOINs and aggregations
- Data consistency and scalability focus

## Tools & Technologies
- MySQL
- SQL (Joins, Aggregations)
- Database Normalization (3NF)

## Project Type
Academic Database Design Project
